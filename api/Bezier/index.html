Bezier

The class used to construct Bézier curves.

Functions​

new​

</>
Bezier.new(
points: {Vector3}--

The array of points

) → BezierCurve

Constructs a new BezierCurve from an array of points. Obviously, you should use two or more points.

fromParts​

</>
Bezier.fromParts(
folder: Instance--

The instance that has the Parts as children

) → BezierCurve

Constructs a new BezierCurve from a folder with Parts. The parts must have a number as their name (1, 2, 3, ...) so the order of the points is clear.

</>
Bezier.random(
start: CFrame | Vector3,--

The first point

points: number,--

The amount of points

distanceBetweenPoints: NumberRange,--

The approximate distance between points

variation: number?--

The smaller the variation, the more it will look like a straight line

) → BezierCurve

Constructs a random BezierCurve. If start is a CFrame, the curve will go in its LookVector's direction, otherwise in a random direction.
